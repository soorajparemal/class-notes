Parameters with a default value must be grouped together in a function.

Variable argument functions(VAR-ARGS function) are functions which accepts any number of parameters.Its denoted with a '*'. Parameters are collected as a tuple. '**' collects values and parameters as a dictionary.

printf() and scanf() are elementary var-args functions in c.

Its easier to use var-args function in pyhgton than in c .
